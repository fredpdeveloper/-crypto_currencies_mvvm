
Crypto ejemplo MVVM
=======================

Esta aplicacion de ejemplo te permitira ver los distintos tipos de monedas virtuales, además de noticias relacionadas
usando arquitectura MVVM, databinding, retrofit, rooms y hilt, más adelante explicare un poco de cada una :D
